Introduction

Welcome! This website demonstrates my use of CanvasMap, which was created by Dr. James Graham for the purpose of generating an open source library to create interactive maps. The purpose of this assignment was to add a map that moves with the page and one that "resizes" and takes up most of the page. The resizable map is intended to be used as an application that provides the target audience with an issue, and communicate what it accomplishes. This site also includes my utilities from my previous site. Feel free to take a look around.
